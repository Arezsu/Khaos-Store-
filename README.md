## ⚙️ INSTALL

```bash
git clone https://github.com/Arezsu/Khaos-Store-.git
cd Khaos-Store-

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
